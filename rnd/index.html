Снежинка

Сбор исторических данных по поисковым запросам

  • Агрегирует абсолютные исторические данные по заданным ключевым словам для wordstat
  • Строит на их базе выгрузку из similarweb, относительный сплит domain/engine/paid/organic
  • Расчитывает абсолютные значения для указанного сплита

Еженидельный компет

Агрегация данных по конкурентным группам клиента

  • Осуществляет регулярную выгрузку данных из similarweb
  • Интерполирует недостающие временные интервалы для мобильного трафика
  • Формирует выгрузку для дальнейшей обработки в BI

Перфоманс калькулятор

  • Осущесвляет расчет оптимального распределения бюджета между рекламными каналами
  • Подбирает каналы для наиболее эффективнного достижения заданных показателей
  • Расчитывает максимальную возможную прибыль для заданных канналов

Скриншотер

Автоматизация подготовки скриншот-отчетов

  • Создает полный архив указанной веб страницы
  • Вносит в верстку необходимые правки
  • Делает скриншет

Кластеризация

Кластеризация потребительской корзины

  • Обработка ~ 15М покупок
  • Понижение размерности данных и кластеризация
